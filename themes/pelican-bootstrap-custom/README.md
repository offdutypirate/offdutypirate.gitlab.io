# pelican-bootstrap3-custom

This is a Bootstrap 3 theme for Pelican, originally developed by
[DandyDev](https://github.com/DandyDev).  It is now part of the [Pelican Themes](https://github.com/getpelican/pelican-themes) project. Pelican-bootstrap3 is compatible with
Pelican 3.3.0 and higher.

Modifications have been made and the blog stylesheet from Bootstrap Examples have been used.
